## Features
**boldtemple Game Engine** it's a simple and lightweight 2d game framework:
* Sprite animation
* Sound effects
* Simple AABB collision detection
* Camera controlling
* Widget system for making GUI
* Resource manager
 
**Soon:**
* Particle system
* Vector collision detection / Box2d integration
* Scene manager
* Script engine integration
